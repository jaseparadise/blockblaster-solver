Solver notes

  • If a piece does not fit immediately, the solver tries using the other pieces first to create space.
  • It evaluates every order of the 3 pieces and every legal placement sequence.
  • When several sequences are legal, it favors more clears, fewer remaining blocks, fewer holes, and better future flexibility.
  • This version is fully usable for solving. Automatic screenshot recognition is the next upgrade.
